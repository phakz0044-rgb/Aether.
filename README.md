# Aether

Aether is a modern AI assistant inspired by ChatGPT.

It was built to provide intelligent conversations while keeping chats, projects and users organized.

## Features

- AI chat
- Chat history
- Multiple projects
- FastAPI backend
- React frontend
- SQLAlchemy ORM
- SQLite (PostgreSQL ready)
- Groq API integration
- REST API

## Tech Stack

### Backend

- FastAPI
- SQLAlchemy
- Pydantic
- Uvicorn

### Frontend

- React
- TypeScript
- TailwindCSS

### AI

- Groq API
- Llama 3.3 70B
